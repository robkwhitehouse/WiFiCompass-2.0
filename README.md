# WiFiCompass-2.0
Transmits a Compass heading over WiFi - NMEA 0183 HDM format

This version adds code support for the Sparkfun BNO-085 IMU module. The previous version only supported the Devantech CMPS-14 module.
See previous version README for a description of functionality. 
This is an ESP32 Arduino sketch.
The .ino file is fully commented
